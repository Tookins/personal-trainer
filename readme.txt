readme file for personal trainer website

this website contains an application that allows the user to create workouts, save them, and add them to a database. it will also have the ability to generate workouts from a database of exercises. the purpose of the site is to provide users with an easy way to design their own training plan and keep track of their training over time. the ultimate goal of the application is to automate the process of physical training in a way that can be easily tailored to the users needs using an intelligent program design similar to a recommendation system, where the user can give input about how they want to train and the application will provide them with the next workout that they should do.

features:
reader page
writer page
list page
recommender algorithm
